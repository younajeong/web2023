  • table-layout: 테이블의 셀 너비를 조정하는 방식을 설정하는 CSS 속성입니다
  • top: 요소의 상단 위치를 설정하는 CSS 속성입니다
  • transform: 요소의 변환(translation, rotation, scaling) 효과를 정의하는 CSS 속성입니다.
  • transform-origin: 요소의 변환 기준점을 설정하는 CSS 속성입니다.
  • transform-style: 3D 변환된 자식 요소가 3D 공간에 있는 것처럼 렌더링되는지 여부를 설정하는 CSS 속성입니다.
  • transition: CSS 속성의 변화를 부드럽게 전환하는 효과를 설정하는 CSS 속성입니다
  • transition-delay: 속성 변화가 시작되기 전에 대기하는 시간을 설정하는 CSS 속성입니다.
  • transition-duration: 속성 변화가 일어나는 지속시간을 설정하는 CSS 속성입니다.
  • transition-property: 전환 효과를 적용할 CSS 속성을 지정하는 CSS 속성입니다. 이 속성에 지정한 속성이 변경될 때, 전환 효과가 적용됩니다.
  • transition-timing-function: 전환 효과의 타이밍 함수를 지정하는 CSS 속성입니다.
  • unicode-bidi: 언어 흐름과 문자 흐름의 방향을 설정하는 CSS 속성입니다
  • user-select: 사용자 선택 가능 여부를 지정하는 CSS 속성입니다.
  • vertical-align: 요소의 수직 정렬 방식을 설정하는 CSS 속성입니다.
  • visibility: 요소의 표시 여부를 설정하는 CSS 속성입니다
  • word-spacing: 단어 사이의 간격을 설정하는 CSS 속성입니다.
  • word-wrap: 텍스트 줄 바꿈을 설정하는 CSS 속성입니다.
  • z-index: 요소의 쌓이는 순서를 설정하는 CSS 속성입니다
