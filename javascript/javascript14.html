01. 퀴즈 이펙트 : 정답 확인하기 유형

정답 확인 버튼을 누르면 정답이 나오는 효과입니다.

  • 데이터 저장: 변수
  • querySelector() : 문서에서 특정 CSS 선택자로 요소를 찾는 DOM 메서드입니다.
  • innerText : DOM 요소의 텍스트 내용을 가져오거나 설정하는 데 사용되는 속성입니다.
  • style.속성 = "속성값" : 자바스크립트에서 CSS스타일을 표현하는 방법입니다.
  • classlist. : 요소의 클래스를 추가하거나 제거하여 스타일을 설정하는데 사용됩니다.
  • addEventListener : 자바스크립트에서 이벤트를 처리하는 함수입니다.

02. 퀴즈 이펙트 : 주관식 확인하기 유형

정답을 적으면 맞는지 확인해주는 효과입니다.

  • 데이터 저장: 변수
  • 데이터 제어하기: if문
  • trim() : 함수 문자열의 양 끝에 있는 공백(스페이스, 탭, 개행 등)을 제거하는 함수
  • textContent : DOM요소의 텍스트 컨테츠를 나타내는 요소입니다.
  • value : 값을 나타내는 속성입니다.input 요소에서는 입력 필드의 현재 값을, select 요소에서는 선택된 옵션의 값을, textarea 요소에서는 텍스트 영역의 텍스트 값을 나타냅니다.
