    06. 데이터 이펙트 : for문을 이용해서 100칸 테이블 만들기

    for문을 이용해서 100칸 테이블 만드는 예제입니다.

    let table = "<table border=1>";

for(let i=1; i<=10; i++){
    table += "<tr>"

    for(let j=1; j<=10; j++){
    table += "<td>" + j + "</td>"
    }

    table += "</tr>" 
}
table += "</table>"; 

document.write(table); 

//1부터 100까지 써있는 100칸짜리 표가 만들어집니다.
