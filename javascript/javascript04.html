05. 중첩 if

if문 안에 중첩으로 여러 if문이 들어갈 수 있습니다.

{    
    const num = 100;

    if(num == 100){
        document.write("실행되었습니다.(1)");

        if(num == 100){
            document.write("실행되었습니다.(2)");

            if(num == 100){
                document.write("실행되었습니다.(3)");
     
            }
        }            
    }else{
        document.write("실행되었습니다.(4)");
}
결과 확인하기
100일 경우 실행되었습니다(1),(2),(3)
