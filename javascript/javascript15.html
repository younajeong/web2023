02. 퀴즈 이펙트 : 주관식 확인하기 유형

정답을 적으면 맞는지 확인해주는 효과입니다.

  • 데이터 저장: 변수
  • 데이터 제어하기: if문
  • trim() : 함수 문자열의 양 끝에 있는 공백(스페이스, 탭, 개행 등)을 제거하는 함수
